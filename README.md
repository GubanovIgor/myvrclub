# Агрегатор клубов виртуальной реальности
# [myvrclub.ru](https://myvrclub.ru)

## Как запустить:
myvrclub/backend
```
npm install
node app.js
```
myvrclub/frontend
```
npm install
npm run build
npm run start
```
